Многостраничное React-приложение с использованием роутинга, объединяющее несколько мини-приложений (трекеры, заметки, тесты и др.) в одном интерфейсе.

Функционал
Приложение включает несколько независимых модулей:
* Profile — профиль пользователя
* Quiz — тесты/викторина
* Dictionary — словарь
* Habit Tracker — трекер привычек
* Notes — заметки
* Pomodoro Timer — таймер продуктивности
* 🛒 Shopping List — список покупок
* Dark Theme -переключать с тесной на светлую сторону
* Count History - считать клик 
* Countdown - обратный отчет

Навигация
Реализована с помощью React Router
